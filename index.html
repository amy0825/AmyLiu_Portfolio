Enterprise AI Support Assistant

Internal AI chatbot for IT support automation

Project Overview

Developed a private, on-premise AI assistant to help employees resolve IT-related queries using natural language. Integrated Retrieval-Augmented Generation (RAG) architecture with DeepSeek LLM and embedded internal documentation.

My Role

  • Designed and implemented a Flask-based web backend
  • Built vector database for document retrieval (FAISS)
  • Created an intuitive chat UI with JavaScript
  • Ensured full GDPR-compliant on-premise deployment

Tech Stack

Python, Flask, JavaScript, FAISS, DeepSeek LLM, Docker

Impact

  • Reduced internal IT support tickets by 60%
  • Improved employee satisfaction with quicker issue resolution
  • Established a scalable template for enterprise AI integration

Architecture Overview

The system used document embeddings with FAISS for retrieval and passed relevant content to the LLM for generating responses. Everything ran in a secure, internal server environment.
